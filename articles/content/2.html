Markdown to HTML Converter

Конвертер Markdown в HTML с графическим интерфейсом (Swing) и консольной версией.

Поддерживает 15 цветовых тем, три варианта выравнивания текста, таблицы, чек‑листы, блоки кода с тёмной/светлой подсветкой, эмодзи, HTML‑сущности и многое другое.

Требования

  • Java 11 или новее (рекомендуется Java 17/21 LTS)

Быстрый старт

  1. Скачайте нужный вам .java файл:
  • MarkdownToHTMLConverter.java – GUI версия
  • MarkdownToHTMLConverterCLI.java – консольная версия
  1. Поместите файл в любую папку.
  2. Создайте подпапку resources и положите туда ваши .md файлы.
  3. Запустите программу:

GUI версия

java MarkdownToHTMLConverter.java

CLI версия

java MarkdownToHTMLConverterCLI.java

Следуйте инструкциям в терминале: выберите язык, папку, имя файла, выравнивание, тему и т.д.

Результат – в папке converted

Структура проекта

MDToHtmlConverter/
├── MarkdownToHTMLConverter.java        # GUI-версия
├── MarkdownToHTMLConverterCLI.java     # CLI-версия
├── resources/                          # исходные .md файлы (создаётся автоматически)
│   └── example.md
└── converted/                          # результат .html (создаётся автоматически)
    └── example.html

Возможности

  • Заголовки: h1-h6
  • Выделение текста: жирный, курсив, зачёркнутый
  • Списки: нумерованные, маркированные, вложенные
  • Чек-листы: - [x] / - [ ]
  • Цитаты: >
  • Таблицы: в Markdown-синтаксисе
  • Блоки кода: с подсветкой (язык указывается после ```)
  • Инлайн‑код: code
  • Ссылки: в том числе внутренние #, относительные, email, автоссылки
  • Изображения: alt, url
  • Горизонтальные линии: ---, ***
  • Эмодзи: 😊 → 😊
  • HTML‑сущности: ̄, © и др.
  • Прямой HTML внутри Markdown: опционально

Настройки (GUI)

  • Язык: Русский / English (мгновенное переключение)
  • Выравнивание: Левое, Центр, Правое
  • Темы: 15 цветовых схем (Светлая, Тёмная, Сепия, Лесная, Океан, Космос, Кофе, Мятная, Лаванда, Закат, Арктика, Пустыня, Вишня, Ночь, Рассвет)
  • Разрешить HTML‑теги и скрипты – даёт возможность вставлять произвольный HTML/JS

Настройки (CLI)

Все параметры задаются через консольные вопросы:

  • Язык
  • Путь к папке с .md
  • Имя файла без расширения
  • Выравнивание
  • Тема (1-15)
  • Разрешение HTMl
