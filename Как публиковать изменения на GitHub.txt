+ приложение
- Terminal
| github.com

------------------

- 1) gh repo clone <repository> (Клонировать репозиторий)
- 2) git status (проверить статус репозитория)
- 3) Внести изменения
- 4) git add . (зарегистрировать изменения)
- 5) Сделать коммит (git commit -m "комментарий")
- 6) git push (Сделать Push)
| 7) Заходим на github.com
| 8) Принимаем Push, делаем всё по подсказкам.

Полезные комманды:
[git pull origin ВЕТКА] - обновить локальную ветку
[git log --oneline --all --graph] - посмотреть все коммиты
[git fetch origin] [git reset --hard origin/main] - принудительно обновить локальные ветки
