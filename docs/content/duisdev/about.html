DUISDev

Официальный сайт команды: Open Source проекты, контент и инструменты для веб-разработки.

Что это

DUISDev — статический сайт на HTML/CSS/JS без серверного бэкенда. Разделы (блог, статьи, видео, документация) подключают общие модули шапки и меню из папки modules/.

Как устроена документация

Вы читаете раздел DUISDev — навигация для разработчиков по всему проекту, включая видеоплатформу. Отдельная документация Page Builder описывает CSS-фреймворк в dependencies/PageBuilder/.

Главные точки входа

  • index.html — главная страница
  • modules/menu.html — меню сайта (редактируется один раз для всех страниц)
  • js/include-modules.js — подгрузка модулей на страницах
