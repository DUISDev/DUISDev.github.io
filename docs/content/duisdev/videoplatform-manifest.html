manifest.json

Каталог видеоплатформы: каналы, плейлисты и ролики.

Авторы (каналы)

{
  "name": "DUISDev",
  "handle": "@duisdev",
  "avatarColor": "#0055ff",
  "avatarUrl": "avatars/duisdev.png",
  "description": "Описание канала"
}

Id создаётся из handle. Фото — необязательное поле avatarUrl.

Плейлисты

{
  "title": "С чего начать",
  "description": "Описание",
  "author": "@duisdev",
  "coverUrl": "covers/start.jpg",
  "videos": ["Чёрный экран"]
}

Без обложки — градиент без текста. Количество видео показывается в подписи карточки.

Видео

{
  "title": "Чёрный экран",
  "author": "@duisdev",
  "playlists": ["С чего начать"],
  "source": { "type": "file", "url": "videos/example.mp4" }
}

Тип embed — для внешнего iframe. Id ролика генерируется из названия.

Страница канала

Вкладка «Все» — только видео автора. «Плейлисты» — только карточки плейлистов. Ссылки: index.html?author=duisdev, ?playlist=....
