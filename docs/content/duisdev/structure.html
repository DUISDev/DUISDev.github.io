Структура проекта

Карта папок DUISDev — что где лежит и за что отвечает.

Корень сайта

DUISDev/
ПутьНазначение
index.htmlГлавная: hero, разделы, команда
css/Общие стили: шрифты, меню, footer, страницы разделов
js/Скрипты: include-modules, страницы разделов
modules/HTML-модули: logo, menu, footer, hero, features, team
images/Изображения главной и общие ресурсы
dependencies/Page Builder и другие зависимости

Разделы

ПапкаОписание
blog/Блог, посты, админка, сборщик build-index.java
articles/Статьи по категориям, manifest.json
videoplatform/Видеокаталог, manifest.json, watch.html
docs/Документация (этот раздел)

Страница в подпапке

Укажите корень сайта для модулей и скриптов:

<html lang="ru" data-site-root="../">
<head><meta charset="UTF-8"></head>
<body>
  <header data-duisdev-module="menu"></header>
  <main>Контент раздела</main>
  <footer data-duisdev-module="footer"></footer>
  <script src="../js/include-modules.js" defer></script>
</body>
</html>

Для главной data-site-root="", для вложенных разделов — ../ или глубже.
