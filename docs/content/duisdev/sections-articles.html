Статьи

Каталог гайдов по категориям с просмотром в iframe.

Файлы

  • articles/index.html — каталог
  • articles/manifest.json — категории и список статей
  • articles/content/*.html — содержимое статей
  • js/articles-page.js, js/article.js — логика и высота iframe
  • css/articles-page.css, css/article-content.css — оформление

Как добавить статью

1. Положите HTML в articles/content/.
2. Добавьте запись в категорию в manifest.json:

{ "title": "Название", "file": "3.html" }

Id создаётся автоматически из имени файла.
