Блог

Посты, документы, админка и сборка индекса.

Структура

blog/
  • index.html, posts/postN.html — генерируются сборщиком, не править вручную
  • admin/index.html — админка и предпросмотр
  • build-index.java — сборка каталога постов
  • posts/ — исходники постов и вложения

Как добавить пост

Создайте HTML в blog/posts/, затем запустите javac build-index.java && java build-index в папке blog/.

Для документов в посте используйте класс post-document и ссылку на файл в posts/assets/.
