Локальный запуск

Сайт статический, но для модулей и manifest нужен HTTP.

Зачем сервер

fetch() для manifest.json, модулей и iframe не работает при открытии файлов напрямую (file://).

Python

cd DUISDev
python3 -m http.server 8080

Откройте http://localhost:8080/

Node.js

npx serve .
# или
npx http-server -p 8080

GitHub Pages

Корень репозитория — корень сайта. Пути data-site-root настроены для вложенных папок.
