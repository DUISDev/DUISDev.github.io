Раздел «Документация»

Каталог документаций и просмотр с оглавлением.

Файлы

  • docs/index.html — карточки всех документаций
  • docs/read.html — чтение с боковым оглавлением
  • docs/manifest.json — список документаций и страниц
  • docs/content/ — HTML-страницы
  • js/docs-data.js, docs-hub.js, docs-reader.js
  • css/docs-page.css, css/docs-content.css

Как добавить страницу

Создайте HTML в docs/content/... и добавьте в нужную документацию в manifest:

{ "id": "new-page", "title": "Новая страница", "file": "duisdev/new-page.html" }

После правки manifest.json скопируйте его содержимое в MANIFEST внутри js/docs-data.js.
