Page Builder

CSS-фреймворк DUISDev для быстрой вёрстки: темы, цвета, размеры, анимации и стиль custom.

Идея

Набор готовых utility-классов вместо написания CSS с нуля. Подключите один файл loader.css — он импортирует все модули.

Где лежит

dependencies/PageBuilder/
  • css/loader.css — точка входа
  • css/colors/ — фон и текст
  • css/elements/ — темы, размеры, свойства, custom
  • css/animations/ — loading, constant, hover
  • index.html — пример страницы

Имена классов

Используйте kebab-case: loading-fade-in, hover-pulse-animation, light-theme. Несколько классов комбинируются на одном элементе.
