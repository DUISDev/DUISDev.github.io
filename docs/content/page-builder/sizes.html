Размеры

Фиксированные width/height через классы size-Npx.

Классы

<div class="size-100px bg-blue center">100×100</div>
<div class="size-240px bg-teal center">240×240</div>

Файл css/elements/size.css — классы от size-1px до больших значений (тысячи строк, шаг 1px).

С center

Класс center из properties.css — flex по центру. Удобно для квадратов и иконок.
