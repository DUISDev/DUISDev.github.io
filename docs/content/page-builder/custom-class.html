Класс custom

Единый стиль DUISDev для кнопок, ссылок, медиа и текстовых блоков.

Назначение

Класс custom добавляет белый фон, синюю рамку, скругление и лёгкое свечение. Работает на разных тегах — правила в css/elements/custom.css.

Примеры

<button class="custom">Кнопка</button>
<a class="custom" href="#">Ссылка</a>
<p class="custom">Текстовый блок</p>
<img class="custom" src="photo.png" alt="">

button.custom a.custom p.custom img.custom video.custom

С hover-анимациями

Комбинируйте с классами из hover-animations.css: custom hover-pulse-animation.
