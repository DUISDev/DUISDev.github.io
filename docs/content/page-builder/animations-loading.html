Анимации при загрузке

Появление элементов при открытии страницы — один раз, без бесконечного цикла.

Как использовать

Добавьте класс на элемент или на <body>. Файл: css/animations/loading-animations/loading-animations.css.

<body class="loading-fade-in-up">
<section class="loading-slide-up">...</section>

Основные классы

loading-fade-in loading-fade-in-up loading-fade-in-down loading-slide-up loading-zoom loading-scale-in

Есть варианты с направлением: loading-fade-in-left, loading-slide-screen-up и др.

Демонстрация — обновите iframe, чтобы увидеть анимацию снова
